# Art Generation Tools

Put repeatable asset-generation scripts here.

The project rule is that Codex owns art production and import. When an asset can
be generated by code, keep the script here and write the runtime output to
`public/assets/generated/`.

Examples:

- Generate SVG/PNG card icons.
- Generate spritesheets for fire, sparks, healing, or impact effects.
- Export simple procedural GLTF/GLB models.
- Batch optimize generated textures.
